双网卡实现内外网同时访问

2022年07月06日 Guestbook

Windows

1.展示路由
route print
2.添加路由
route add 0.0.0.0 mask 0.0.0.0 192.168.110.1（想要使用的网络的网关） -p
route add 10.16.0.0 mask 255.255.255.0 10.16.128.1（想要使用的网络的网关） -p
3.删除路由
route delete 网关。

Mac

route -n add -net 10.16.88.66 10.16.128.1
